## 使用方法

通过以下命令在游戏中切换模式：

- `/gms`：切换到生存模式
- `/gmsp`：切换到观察者模式

## 权限

- `sakura.gamemode.survival`：执行 `/gms` 命令的权限
- `sakura.gamemode.spector`：执行 `/gmsp` 命令的权限
